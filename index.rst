介绍
==================================================

    Script: A writing; a written document. [Obs.]
    
    --Webster's Dictionary, 1913 ed.

Shell 是指令解释器。它不仅仅是用户与操作系统内核的封装层，它更是一种相对强大的编程语言。一个 Shell 程序，又叫脚本，是一种易用的，由系统调用、工具、和编译的二进制文件「连接」而成的程序建造工具。Shell 脚本里可以执行几乎整个 UNIX 指令和工具的集合。如果那还不够的话，像测试和循环语法等内部 Shell 命令赋予脚本额外的能力和灵活度。Shell 脚本尤其适合于系统管理任务及其他不需紧密架构语言全部机能的重复劳作。
